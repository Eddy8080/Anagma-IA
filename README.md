# Anagma IA - Inteligência Contábil e Gestão de Ideias

Anagma IA é uma plataforma avançada para gestão, refinamento e exploração de ideias, especialmente desenhada para o setor contábil. Utilizando inteligência artificial, o projeto integra o conhecimento técnico especializado à cultura organizacional da empresa para potencializar a criatividade e a organização de novos processos e conceitos. O projeto está atualmente em fase ativa de desenvolvimento.

## 🚀 Sobre o Projeto

O Anagma IA vai além de um simples repositório; ele atua como um ecossistema onde ideias brutas são processadas, categorizadas e enriquecidas através de capacidades cognitivas computacionais. A plataforma permite que os colaboradores registrem suas visões e recebam insights assistidos para estruturação e viabilidade, sempre em conformidade com as práticas contábeis e os valores específicos da organização.

### Principais Funcionalidades (Em Desenvolvimento)

- **Gestão de Ideias Contábeis:** Interface intuitiva para criação, edição e acompanhamento de propostas de melhoria em processos fiscais, contábeis e consultivos.
- **Assistente Cognitivo Alinhado:** Integração com motor de IA treinado para fornecer respostas e análises que refletem a cultura e o tom de voz da empresa.
- **RAG (Retrieval-Augmented Generation):** Exploração inteligente de contextos baseada na base de conhecimento interna e normas contábeis vigentes.
- **Painel Administrativo:** Controle total sobre usuários, perfis e visibilidade global de ideias, garantindo a governança da informação.
- **Arquitetura Escalável:** Construído com Django para garantir segurança, robustez e a confidencialidade exigida pelo setor contábil.

## 🛠️ Tecnologias Principais

- **Backend:** Python / Django
- **IA/ML:** Motores de Inferência Local e Vector Stores (foco em privacidade)
- **Banco de Dados:** SQLite (Desenvolvimento) / PostgreSQL (Produção)
- **Frontend:** HTML5, CSS3, JavaScript (Vanilla)

## 📂 Estrutura do Projeto

O projeto está organizado para separar claramente a lógica de negócio da inteligência artificial:

- `/src/core`: Lógica principal da aplicação e gestão de usuários.
- `/src/chat_ai`: Motores de IA, integração RAG e processamento de linguagem natural.
- `/assets`: Armazenamento de vetores e recursos de inteligência organizacional.

## 🚧 Status do Desenvolvimento

Este projeto é um **Work in Progress (WIP)**. Funcionalidades de interface administrativa, autenticação e integração inicial de IA já estão presentes, enquanto módulos de análise profunda de processos contábeis e expansão do RAG cultural estão em fase de implementação.

---

**Nota:** Anagma IA prioriza a execução local de inteligência para garantir a privacidade absoluta e a soberania dos dados estratégicos e contábeis da organização.
